Gridonkulous: a tiny fluid grid system for Less

Gridonkulous is a simple fluid grid system for Less. You can use it with semantic class names or numerical class names, your choice. It's mobile-first and easy to combine with media queries to create a fully responsive grid, starting from any number of columns. Download it on Github.

Gridonkulous was inspired by the Semantic Grid System and was originally based on the Dead Simple Grid.
